Image Text Processor v1.0.0
==========================

Instructions:
1. Extract all files to a folder
2. Place your images in the 'images' folder
3. Place your Excel files in the 'text' folder
4. Double-click image-text-processor.exe to run the program
5. Processed images will appear in the 'result' folder

Supported file types:
- Images: .jpg, .jpeg, .png, .gif, .webp
- Text: .xlsx, .xls
